## Описание реализации
Приложение содержит:
- поле для ввода исходного текста;
- поле для вывода результата;
- таблицу кодов;
- кнопки шифрования, дешифрования, загрузки таблицы по умолчанию, добавления и удаления строк, очистки полей;
- область вывода ошибок;
- область вывода сообщения о состоянии.

В программе реализованы:
- отдельный метод шифрования;
- отдельный метод дешифрования;
- проверка пустого текста;
- проверка пустых ячеек таблицы кодов;
- проверка дублирующихся слов;
- проверка дублирующихся кодов;
- обработка отсутствующих слов и кодов;
- всплывающие подсказки;
- XML-комментарии


## Средства отладки
При отладке программы использовались встроенные средства Visual Studio:
